How to clone this project Perfectly !!

1.clone this project into your folder
2.Open it using any IDE you want
3.Get into your frontend folder by Type "cd frontend" in your IDE terminal
4.type "npm install"
5.Get into your backend folder by type "cd ../backend" in your IDE terminal
6.type "pip install -r requirements.txt"
7.Setup .env by copy .env.example and add your own API keys
